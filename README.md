# api-flask-mysql
Crud using pyhton with flask and MySQL database



In the project folder run:

 # Step-1
    
    # pip install -r requirements.txt
    
 # Step-2
 
    # flask run
    
 # server access

    # Running on http://127.0.0.1:5000/
